RLE解压缩库示例

功能说明

这是一个基于Go代码逻辑的纯JavaScript RLE解压缩库，支持以下格式：

  • 15位 (RGB555) - 5位红、5位绿、5位蓝
  • 16位 (RGB565) - 5位红、6位绿、5位蓝
  • 24位 (BGR) - 8位蓝、8位绿、8位红
  • 32位 (BGRA) - 8位蓝、8位绿、8位红、8位透明

测试解压缩功能

使用示例代码

// 创建解压缩器实例
const decompressor = new RLEDecompressor();

// 解压缩15位RGB555数据
const output = new Uint8Array(width * height * 4); // RGBA格式
const success = decompressor.bitmapDecompress15(
    output,           // 输出缓冲区
    outputWidth,      // 输出宽度
    outputHeight,     // 输出高度
    inputWidth,       // 输入宽度
    inputHeight,      // 输入高度
    compressedData    // 压缩的输入数据
);

// 将结果绘制到Canvas
const canvas = document.getElementById('canvas');
const ctx = canvas.getContext('2d');
const imageData = ctx.createImageData(width, height);
imageData.data.set(output);
ctx.putImageData(imageData, 0, 0);

技术特性

  • 字节序：使用小端序（低字节在前），符合RDP协议
  • 颜色转换：自动将RDP的BGR/BGRA格式转换为Web标准的RGBA格式
  • 性能优化：使用TypedArray提高性能，支持大图像处理
  • 错误处理：完善的错误检查和异常处理
  • 兼容性：支持Node.js和浏览器环境
